## ðŸ“‹ Step-by-Step Replacement Process

### Phase 1: Characters & Animations (Mixamo)

1. **Sign up at Mixamo** (free)
2. **Download 6 characters** (see list above)
3. **Download 11 animations** (see list above)
4. **Convert to GLB** (if needed):
   ```bash
   # Install gltf-pipeline
   npm install -g gltf-pipeline
   
   # Convert FBX to GLB
   gltf-pipeline -i character.fbx -o character.glb
   ```
5. **Replace files**:
   ```bash
   cp ~/Downloads/male_base.glb frontend/public/models/characters/
   ```

### Phase 2: Robots & Environment (Poly Pizza)

1. **Visit Poly Pizza**
2. **Search and download** 9 robots + 7 environment pieces
3. **Download as GLB**
4. **Replace files**:
   ```bash
   cp ~/Downloads/*.glb frontend/public/models/robots/
   ```

### Phase 3: Textures (Poly Haven)

1. **Visit Poly Haven**
2. **Download 31 textures** (4K PNG)
3. **Resize if needed**:
   ```bash
   # Install ImageMagick
   sudo apt install imagemagick
   
   # Resize to 2K
   convert input.png -resize 2048x2048 output.png
   ```
4. **Replace files**:
   ```bash
   cp ~/Downloads/*.png frontend/public/textures/
   ```

### Phase 4: Sounds (Freesound)

1. **Sign up at Freesound**
2. **Download 8 sounds** (WAV format)
3. **Replace files**:
   ```bash
   cp ~/Downloads/*.wav frontend/public/sounds/
   ```

### Phase 5: Images & Icons

1. **Download from Unsplash/Pexels**
2. **Download icons from Flaticon**
3. **Replace files**:
   ```bash
   cp ~/Downloads/*.jpg frontend/public/images/
   cp ~/Downloads/*.svg frontend/public/icons/
   ```

### Phase 6: Fonts (Google Fonts)

1. **Download Orbitron from Google Fonts**
2. **Extract WOFF2 files**
3. **Rename and replace**:
   ```bash
   cp ~/Downloads/Orbitron-Regular.woff2 frontend/public/fonts/game_font_regular.woff2
   cp ~/Downloads/Orbitron-Bold.woff2 frontend/public/fonts/game_font_bold.woff2
   ```
